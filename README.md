# GT Tires Client Mini App

Готовий фронтенд-дизайн клієнтського Telegram Mini App для GT Tires.

## Що всередині

- `index.html` — головний файл
- `styles.css` — дизайн
- `app.js` — навігація і тестова логіка
- `manifest.json` — PWA/мобільний опис

## Як запустити

Просто відкрий `index.html` у браузері.

## Як залити на GitHub Pages

1. Завантаж всі файли в репозиторій.
2. В GitHub відкрий Settings → Pages.
3. Source: Deploy from branch.
4. Branch: main / root.
5. Збережи.

## Telegram Mini App

Після публікації сайту URL можна вставити в BotFather як Web App URL.
